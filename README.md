# express-sequelize-swagger-starter

## Description

This repository, hope can help u developing backend faster than before becuase this project already structural. just need to settingup some.

Using Technologies :
- Express JS
- Sequelize
- Api Docs : Swagger.
- etc

## How to use

- clone this repository 

  `git clone git@github.com:raindev32/express-sequelize-swagger-starter.git`

- Make mysql database

- Setting config file in config/config.json

- Setting the `.env` file from `.env.example`

- Install the dependencies

  `npm install ` OR `yarn install `

- Run development

  `npm start-dev` OR `yarn start-dev`
